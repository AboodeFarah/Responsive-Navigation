# Responsive Navigation Bar Project

## Overview

This project is a Responsive Navigation Bar built using HTML, CSS, and JavaScript. The navigation menu automatically adapts to different screen sizes, providing a better user experience on both desktop and mobile devices.

On larger screens, the navigation links are displayed horizontally across the top of the page. On smaller screens such as smartphones and tablets, the navigation links are hidden behind a hamburger menu icon. When the icon is clicked, the menu expands smoothly, allowing users to access all navigation links.

## Features

* Responsive design for desktop, tablet, and mobile devices.
* Clean and modern navigation layout.
* Hamburger menu for mobile screens.
* Smooth menu opening and closing animation.
* Easy-to-customize navigation links.
* Lightweight and beginner-friendly code structure.

## How It Works

The navigation bar contains a website title and a set of navigation links. On desktop screens, all links are visible at all times. When the screen width becomes smaller, the navigation links are hidden and replaced with a hamburger menu icon.

When the user clicks the hamburger icon, the menu expands to display all navigation options. Clicking the icon again closes the menu. This behavior improves usability on smaller devices by saving screen space while keeping navigation accessible.
